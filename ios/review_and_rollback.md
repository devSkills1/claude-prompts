### 回滚触发条件（满足任一则考虑回滚）

- 崩溃率上升 > 0.1%

- 监控告警触发关键指标
- 发现严重安全漏洞
- App Store 审核被拒

### 回滚方案

**Git 回滚命令：**
```bash
# 方案 1：revert（推荐，保留历史）
git revert <commit_hash>

# 方案 2：reset（谨慎使用，会丢失历史）
git reset --hard <commit_hash>
```
